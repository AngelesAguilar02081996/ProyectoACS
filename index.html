Adopción de mascotas

Se requiere una aplicación web para personas interesadas en la Cuidad de México que deseen adoptar o dar en adopción una mascota.

Esta aplicación permitirá a los usuarios que quieran adoptar y dar en adopción, capturar sus datos en la plataforma para que queden registrados y almacenados en la página web.

Para ser un usuario en la plataforma “Adopta un Amigo” en primera instancia se le solicitara un pre registro el cual permitirá almacenar la información de cada uno de ellos y saber los miembros que conforman la plataforma web.

Los datos solicitados serán:

  1. Nombre
  2. Apellido Paterno
  3. Apellido Materno
  4. Fecha Nacimiento
  5. Sexo
  6. Correo
  7. Nacionalidad
  8. Contraseña
  9. Validar Contraseña
  10. Fecha de Ingreso
  11. Hora

De esta manera se tendrá con el usuario que ha ingresado a la plataforma un primer acercamiento de qué tipo de usuarios se irán registrando y las peticiones que desean realizar en la plataforma.
